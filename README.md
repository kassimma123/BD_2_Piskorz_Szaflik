# Katarzyna Piskorz i Iga Szaflik
 naszym pomysłem jest zrobienie bazy dach (oracle i python) która realizuje inteligentną spiżarnię restauracyjną 
 - rezerwacja produktów spożywczych przez kucharzy - jezeli nie jest zarezerwowany i nie jest zepsuty to mozna
 - menu - jezeli wybierze ktos danie to rezerwuje automatycznie produtky potrzebne na to danie
 - szef kuchni jest administratorem
 - niewykorzystane produkty pod koniec dnia oddaje się biednym
 - jezeli jakis produkt się konczy to automatycznie jest dodawany do listy zakupów
